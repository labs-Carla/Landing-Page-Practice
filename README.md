# Landing Page Practice

A static landing page created as an HTML and CSS practice to reproduce a typical homepage structure with hero, information, testimonial, and call-to-action sections.

## ✨ Features

- Container-based responsive design
- Header with navigation
- Hero section with highlighted text and image
- Informational cards
- Styled testimonial section
- Call-to-action block
- Footer with copyright

## 🛠️ Technologies used

- HTML5
- CSS3

## 📁 Project structure

- `index.html`: main page structure
- `styles.css`: visual styles and layout
- `README.md`: project documentation

## ▶️ How to view it

1. Open `index.html` in your browser.
2. Alternatively, run a simple local server from the project folder.

Python example:

```bash
python -m http.server
```

Then open `http://localhost:8000` in your browser.

## 🎯 Project goal

Practice HTML and CSS layout by building a classic landing page composition and improving the organization of the visual code.
